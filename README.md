# URL SHORTENER

A full-stack URL Shortener web application built using the MERN stack. This project allows users to convert long URLs into short, shareable links with a clean and responsive user interface.

---

# 🚀 Features

- Shorten long URLs instantly
- Generate unique short links
- Redirect users to original URLs
- REST API integration
- MongoDB database integration
- Responsive frontend UI
- Separate frontend and backend architecture
- Fast and lightweight application

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Axios
- Vite
- CSS

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
